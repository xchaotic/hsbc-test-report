FpML 5.0 Validation Rules - Rules for Pricing and Risk

This is the Pricing and Risk part of the validation rule set accompanying the FpML 5.0 Recommendation. The introductory section in the validation section of the specifications contains background information and documentation for this page.

The rules contained on this page contain links to cut down versions of valid and invalid test cases. The cut down test cases are non-normative and are provided for the purpose of documentation only.
